Chess Sudoku Solver

link to project

Python package implementing efficient combinatorial algorithms.

Fast variant Sudoku solver library using an implementation of Donald Knuth’s DLX algorithm for multiset-multicover problems. Includes module for classic Sudoku and “Chess Sudoku” variants, including optional adjacency constraints. Was used to obtain the first calculation of the number of unique solutions to the “Miracle Sudoku” variant.
